windows-快捷键

1.Ctrl：控制键

ctrl+shift+n:新建文件夹

ctrl+w 关闭当前整个应用

ctrl + n 新建

ctrl + alt + delete：进入windows安全选项界面（内部含有任务管理器）

ctrl + shift + esc：任务管理器

ctrl + shift : 切换输入法

文字处理

ctrl+b:加粗 bond

ctrl+c 复制 copy

ctrl+d：删除 delete

ctrl+e：居中 center

ctrl+f：查找 find

ctrl+h：替换 change

ctrl+z 撤销上一步（文字撤销，也可以恢复刚刚删除的文件）

ctrl+y:反撤销

ctrl+n 新建 n

ctrl+p：打印 print

ctrl+s 保存 save

ctrl + u 下划线 underline

ctrl + I 斜线 sliant

ctrl+v：粘贴

ctrl+x: 剪切

2.Win键

win + ← 窗口往右靠，最左边

win + → 窗口最右边

win + ↑ 窗口放大

win + ↓ 窗口缩小

Win+D 应用程序程序与桌面来回切换

Alt+Tab 实现界面切换

3.浏览器常用快捷键

alt+f4：关闭整个浏览

ctrl+w：关闭一个tab页面

ctrl+t:新建一个浏览器tab页面

ctrl+n:打开一个浏览器

ctrl+shift+t:打开上一次浏览器的页面

ctrl+tab：从左往右切换页面

ctr+shift+tab ：从右往左切换页面

ctrl+数字：直接跳到第几页

ctrl+d：打开收藏

ctrl + h 打开浏览器历史记录
